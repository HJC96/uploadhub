## 프로젝트 개요
Java와 SpringBoot를 이용하여 CRUD 게시판을 구현하였습니다. 댓글달기, 파일 첨부, 게시판 조회 등의 기능을 제공하며, 서버사이드 렌더링 방식(ThymeLeaf)을 사용했습니다.
